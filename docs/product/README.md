# Product Docs

Strategische Produktdokumentation für den POD Shop.

## Ordnerstruktur

```
docs/product/
  README.md              ← Diese Datei
  vision.md              ← Produktvision
  phases.md              ← Phasen-Definition mit Exit-Kriterien
  roadmap.md             ← Aktuelle Prioritäten & Status
  phase-0-mvp-launch/    ← Detaildokumente für Phase 0
  phase-1-theming/       ← Detaildokumente für Phase 1
```

## Konventionen

- **Status-Icons:** `done` / `in-progress` / `open` / `parked`
- **Nummerierung:** `P0.1`, `P0.2` etc. für Prioritäten innerhalb einer Phase
- **Dateinamen:** `01-xxx.md`, `02-xxx.md` in Phase-Subfoldern
- **Datumsformat:** `YYYY-MM-DD`
- **Verlinkung:** Relative Pfade innerhalb `docs/product/`
